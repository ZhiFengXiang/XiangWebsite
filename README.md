# Xiang 个人网站 v1.8.3

一个展示个人作品、技能和经历的现代个人网站，采用深色主题设计，包含丰富的动画效果。

## 项目特点

### 🎨 视觉设计

- **深色主题**：专业的暗色界面设计
- **毛玻璃效果**：Win11风格的导航栏和卡片
- **自定义光标**：三层结构的光标效果
- **响应式设计**：完美适配各种设备

### 🚀 动画效果

- **银河系星空背景**：Canvas实现的动态星空
- **打字机动画**：标题文字逐字出现效果
- **滚动渐显**：元素进入视口时的动画
- **技能烟雾效果**：鼠标跟随的粒子效果
- **流星效果**：随机出现的流星动画
- **图片悬停放大**：项目卡片非线性回弹放大动画

### 📱 功能特性

- **项目展示**：8个精选项目详情
- **技能展示**：动态技能条效果
- **时间线**：个人经历时间线
- **评价轮播**：AI评价卡片轮播
- **摄影作品**：智能设备识别和emoji显示

## 技术栈

### 前端技术

- **HTML5**：语义化标签
- **CSS3**：Flexbox、Grid、动画、渐变
- **JavaScript**：ES6+、Canvas API、Intersection Observer
- **响应式设计**：媒体查询、移动优先

### 设计特色

- **暗色主题**：#090a0d 背景色
- **金色强调**：#ffd700 主色调
- **毛玻璃效果**：backdrop-filter: blur
- **自定义字体**：Inter字体家族
