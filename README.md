# Xiang's Website v1.5.0

个人网站源码，展示摄影作品、项目经历与个人信息。

🔗 在线访问：[xiang.cubic-racing.online](https://xiang.cubic-racing.online)

## 技术栈

- HTML5 + CSS3 + JavaScript（纯原生，无框架依赖）
- [Inter](https://rsms.me/inter/) 字体（Google Fonts）
- Canvas 2D（银河系星云背景 + 流星动画）
- 响应式布局，适配桌面端 / 平板 / 手机

## 项目结构

```
.

├── css/

├── js/

│   └── galaxy.js           # Canvas 银河系星云背景（独立模块）
├── images/

│   ├── p1.jpg - p3.jpg     # 项目封面图

└── README.md
```

## 主要特性

### 视觉设计
- 深色主题，网格线 + 彩色光晕背景（每个板块独立配色）
- 毛玻璃卡片效果（`backdrop-filter: blur`）
- `#now` 板块使用 Canvas 银河系星云背景，含流星动画
- 滚动渐显动画（stagger 交错出现）
- 页面加载淡入动画

### 项目卡片效果（v1.5.0 新增）
- **3D 透视旋转**：卡片默认状态 `rotateY(-15deg)` 向画面内倾斜
- **密集叠加**：5 张卡片通过负边距叠加排列，左侧卡片在上层
- **绿色辉光**：悬停时显示 5 层绿色 `box-shadow`，颜色与 `#projects` 背景色系一致
- **平滑回正**：悬停时卡片从 -15° 平滑过渡到 0°，同时微微上浮
- **右侧阴影增强**：默认状态右侧有额外阴影层，增强立体感
- **遮罩渐变**：悬停动画进行时右侧渐变显示，完成后完整露出
- **450ms 缓动曲线**：使用 `cubic-bezier(.25, .46, .45, .94)` 实现丝滑过渡

### 交互功能
- **自定义光标**：三层结构（内圈即时跟随 + 外圈缓动跟随 + hover 放大），仅桌面端启用
- **3D 卡片倾斜**：鼠标移动驱动 `perspective` + `rotateX/Y` 透视效果，`#now` 板块幅度减弱至 35%
- **光标镜头反色**：时间线和 `#now` 卡片内，跟随鼠标的白色圆形通过 `mix-blend-mode: difference` 反转颜色
- **按钮涟漪**：点击按钮时从点击位置扩散的圆形涟漪
- **导航栏**：向下滚动隐藏、向上滚动显示 + 顶部滚动进度条
- **技能计数**：进入视口时从 0 递增到目标百分比
- **手机端汉堡菜单**：点击展开/收起导航面板

### Canvas 银河背景（galaxy.js）
- 高斯分布星点 + 星云 + 雾气层 + 流星
- IntersectionObserver 控制可见性，离开视口自动暂停
- 窗口 resize 时自动重建（250ms 防抖）
- 移动端自动降级（减少粒子数量）

### 响应式设计
- 桌面端（>960px）：5 列项目卡片叠加布局
- 平板端（≤960px）：5 列卡片，尺寸缩小
- 手机端（≤720px）：汉堡菜单、单列卡片、全宽按钮
- 小屏手机（≤420px）：进一步缩小卡片和间距
- 触控设备：禁用 `backdrop-filter` 以节省 GPU，隐藏自定义光标

## 文件说明

### index.html
主页面文件，包含：
- 页面元数据（UTF-8 编码、视口设置、SEO 描述）
- 网站标签页图标（head-icon.jpg）
- Google Fonts 预连接和字体加载
- HTML 结构：导航栏、9 个内容板块、页脚（ICP 备案）
- CSS 和 JS 文件引用（`defer` 异步加载）

### css/styles.css
所有样式，包括：
- CSS 变量（颜色、阴影、圆角、毛玻璃参数）
- 每个板块独立的背景纹理和光晕伪元素（`::before` / `::after`）
- 组件样式（卡片、按钮、导航、时间线、技能条等）
- 项目卡片 3D 叠加效果和绿色辉光样式
- `#now` 板块毛玻璃卡片 + 光标镜头样式
- 3D 倾斜通用样式（`transform-style: preserve-3d`）
- 动画关键帧（渐显、流光、漂移、涟漪、逐字等）
- 四档响应式媒体查询（960px / 720px / 420px）
- 无障碍支持（`prefers-reduced-motion`）

### js/main.js
交互逻辑，按功能模块编号：
1. Hero 标题逐字动画
2. 交错渐显（stagger reveal）
3. 导航栏隐藏/显示 + 滚动进度条
4. 技能数字计数动画
5. 光标镜头反色效果（事件委托）
6. 3D 卡片倾斜动画（`#now` 板块减弱幅度）
7. 按钮涟漪效果
8. 自定义光标（三层结构 + 按需动画）

### js/galaxy.js
Canvas 银河系星云背景，独立模块（IIFE）：
- 配置参数集中在 `CONFIG` 对象
- 可见性检测（IntersectionObserver）+ 窗口 resize 重建
- 星点按颜色分组批量绘制
- 流星使用 swap-and-pop 管理生命周期

## 快速开始

### 克隆仓库
```bash
git clone https://github.com/ZhiFengXiang/Xiang-s-Website.git
cd Xiang-s-Website
```

### 本地运行
直接在浏览器中打开 `index.html` 文件即可查看效果。

### 部署到服务器
将所有文件上传到支持静态网站的服务器即可。

## 如何修改

### 修改样式
编辑 `css/styles.css`。所有颜色和间距使用 CSS 变量（`:root` 中定义），修改一处即可全局生效。

### 修改交互
编辑 `js/main.js`。代码按功能模块编号注释，每个模块有独立的事件监听器。

### 修改银河背景
编辑 `js/galaxy.js`。顶部 `CONFIG` 对象集中管理所有参数（星点数量、流星间隔、颜色等）。

### 修改内容
编辑 `index.html`。HTML 结构清晰，每个板块有中文注释说明用途。

## 开发说明

1. 多文件结构：HTML 在根目录，CSS 和 JS 分别在 `css/` 和 `js/` 子目录
2. 图片文件放在 `images/` 目录中，文件名需与 HTML 中的引用一致
3. `galaxy.js` 是独立模块，不依赖 `main.js`，可单独替换
4. 修改后在浏览器中测试效果，建议使用 Chrome DevTools 检查
5. 触控设备上自定义光标和 `backdrop-filter` 自动禁用
